loaderって?

babel-loader, ts-loader, vue-loaderとかその辺

moduleのソースコードを変換するもの

stringとして来るソースコードをstring or bufferとしてreturnしてやれば変換が完了する

loaderの処理順

use: [ 'abc-loader', 'def-loader' ]

def-loader => abc-loader

正確な処理順

abc-loader.pitch
=> def-loader.pitch
=> def-loader default
=> abc-loader default

pitchがString or Bufferを返却時

対象のdefaultの処理まで省略される

return "abc"

(省略)

どんなことに使われている?

cache-loader

pitchフェイズでcacheファイルのmtimeと依存ファイルのmtimeを比較、差異がない場合はcacheファイルを読みこみreturn

defaultフェイズで依存ファイルのpathとそのmtime、前のloaderで生成した文字列を保存

mtime … 最終変更時刻 (modify time)

最後にファイルを変更した時刻のこと

正確にはファイルにwrite(2)、またはtruncate(2)した場合に変更されるもの
